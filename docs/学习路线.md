# 学习路线

这份路线按“先能调用模型，再能组织上下文，最后做 RAG”的顺序整理。每个阶段都包含学习目标、对应脚本和检查点。

## 阶段 1：模型调用入门

详细讲解：[阶段 1：模型调用入门](阶段01-模型调用入门.md)

对应脚本：

- `02LangChain访问阿里云通义千问大模型.py`
- `03LangChain访问Ollama本地模型.py`
- `04LangChain的流式输出.py`
- `05LangChain调用聊天模型.py`
- `06LangChain调用Ollama的聊天模型.py`
- `07LangChain消息的简写形式.py`

学习目标：

- 区分普通 LLM 与 Chat Model 的调用方式。
- 理解 `invoke()` 和 `stream()` 的区别。
- 知道本地模型和云端模型各自需要什么配置。

检查点：

- 能独立运行一次云端聊天模型调用。
- 能独立运行一次 Ollama 本地模型调用。
- 能解释流式输出为什么适合聊天应用。

## 阶段 2：Embedding 与相似度

详细讲解：[阶段 2：Embedding 与相似度](阶段02-Embedding与相似度.md)

对应脚本：

- `01[扩展]余弦相似度.py`
- `08LangChain访问阿里云嵌入模型.py`
- `09LangChain访问Ollama的本地嵌入模型.py`

学习目标：

- 理解文本如何变成向量。
- 理解余弦相似度与语义检索的关系。
- 知道 Embedding 模型和聊天模型不是同一种能力。

检查点：

- 能说清楚为什么 RAG 需要 Embedding。
- 能用一句话解释“相似文本在向量空间里距离更近”。

## 阶段 3：提示词模板

详细讲解：[阶段 3：提示词模板](阶段03-提示词模板.md)

对应脚本：

- `10通用提示词模板.py`
- `11FewShot提示词模板.py`
- `12模板类的format和invoke方法.py`
- `13ChatPromptTemplate的使用.py`

学习目标：

- 掌握 PromptTemplate 和 ChatPromptTemplate 的基本写法。
- 理解 Few-shot 示例如何约束模型输出。
- 区分 `format()` 与 `invoke()` 的使用场景。

检查点：

- 能写一个带变量的提示词模板。
- 能写一个包含 system、human 消息的聊天提示词模板。

## 阶段 4：Chain 与 Runnable

详细讲解：[阶段 4：Chain 与 Runnable](阶段04-Chain与Runnable.md)

对应脚本：

- `14Chain的基础使用.py`
- `15[扩展]Python的或运算符的重写.py`
- `16Runnable接口源码查看.py`
- `17StrOutputParser解析器.py`
- `18JsonOutputParser解析器.py`
- `19RunnableLambda的基础使用.py`
- `29RunnablePassthrough的使用.py`

学习目标：

- 理解 LangChain 中 `|` 管道写法的含义。
- 理解 Runnable 是如何统一 prompt、model、parser、retriever 的。
- 掌握常见输出解析器的作用。
- 理解 RunnablePassthrough 如何保留原始输入并组合上下文。

检查点：

- 能画出一个 `prompt | model | parser` 的输入输出流程。
- 能解释为什么 Retriever 可以放进 Chain 里。

## 阶段 5：会话记忆

详细讲解：[阶段 5：会话记忆](阶段05-会话记忆.md)

对应脚本：

- `20临时会话记忆.py`
- `21长期会话记忆.py`

学习目标：

- 理解临时记忆和长期记忆的区别。
- 知道聊天历史如何参与下一轮提示词构建。

检查点：

- 能让模型基于前文继续回答。
- 能找到长期记忆保存的位置。

## 阶段 6：文档加载与分割

详细讲解：[阶段 6：文档加载与分割](阶段06-文档加载与分割.md)

对应脚本：

- `22CSVLoader的使用.py`
- `23JSONLoader的使用.py`
- `24PyPDFLoader的使用.py`
- `25TextLoader和文档分割器.py`

学习目标：

- 掌握 CSV、JSON、PDF、TXT 的基本加载方式。
- 理解 Document 对象的 `page_content` 和 `metadata`。
- 理解文档分割为什么会影响检索质量。

检查点：

- 能加载一种结构化数据和一种非结构化文档。
- 能解释 chunk size 和 overlap 的作用。

## 阶段 7：向量存储与检索

详细讲解：[阶段 7：向量存储与检索](阶段07-向量存储与检索.md)

对应脚本：

- `26内存向量存储.py`
- `27外部向量持久化存储.py`

学习目标：

- 理解内存向量库和持久化向量库的区别。
- 掌握添加、删除、相似度搜索的基本流程。
- 知道 Chroma 持久化目录的作用。

检查点：

- 能向向量库写入文档。
- 能使用相似度搜索取回相关文档。

## 阶段 8：RAG 检索问答

详细讲解：[阶段 8：RAG 检索问答](阶段08-RAG检索问答.md)

对应脚本：

- `28向量检索构建提示词.py`
- `29RunnablePassthrough的使用.py`

学习目标：

- 理解 RAG 的完整链路：用户问题 -> 检索 -> 组装提示词 -> 模型回答。
- 理解 Retriever、Prompt、Model、Parser 的协作关系。
- 能识别“回答来自参考资料”与“模型自行发挥”的区别。

检查点：

- 能用自己的数据做一个最小 RAG 问答。
- 能说明 RAG 相比直接问模型的优势。

## 建议记录方式

每学完一个阶段，记录四件事：

- 本阶段解决了什么问题。
- 关键类或函数有哪些。
- 输入和输出分别是什么。
- 自己遇到的报错或疑问是什么。
